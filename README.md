# Excel-Project---Coffee-Shop-Sales-Data
Excel project analyzing Coffee Shop sales using formulas, pivot tables, charts, and an interactive dashboard.

**Overview**
- This project presents a complete analysis of Coffee Shop Sales Data using Microsoft Excel.
The dashboard provides insights into monthly sales, category-wise performance, revenue trends, city-wise contribution, weekday transactions, and overall profit analysis.
It is designed to help understand business performance and support decision-making.

**Objectives**
- To clean, organize, and analyze raw coffee shop sales data using Excel.
- To build an interactive dashboard with slicers for Month, Year, and City.
- To identify top-performing categories, cities, and months.
- To understand customer buying patterns based on weekday transactions.
- To calculate total transactions, quantity sold, revenue, and profit.
- To visualize yearly and monthly revenue & profit trends.

**Tools & Skills Used**
**Tools**

**Skills / Techniques**

- Basic Excel formulas (SUM, COUNT, AVERAGE, Profit calculations)

- Highest revenue comes from New York and Los Angeles.
- Top-performing categories include Desserts, Sandwiches, and Specialty Drinks.
